## 🔒 ADMIN DASHBOARD ("/admin")
Create a secure admin panel (username/password auth stored in Replit DB). It should include:

### 🔍 **Dashboard View**
- Total leads today / this week
- New leads list (name, phone, city, issue, time)
- Filter by zip, time, issue type

### 🧰 **Job Tracker**
- View all active/completed jobs
- Mark job as "In Progress" or "Done"
- Add notes, photos, pricing
- Upload before/after images (stored in Replit DB)

### 📞 **Messaging & Smart Reply Tools**
- Auto-SMS & Email to new leads using **Twilio API**
- Quick-reply templates ("On the way", "Here's your quote", etc)
- **Smart AI Response Assistant**:
  - Use **Together AI (Mixtral or Mistral)** to:
    - Analyze breakdown description
    - Suggest best reply message
    - Recommend fair pricing
  - Show AI suggestion in the UI with a "Use This" button

### 🗺️ **Live Map + Location Tracker**
- Use **Google Maps API** to:
  - Auto-detect job locations
  - Pin new breakdowns on the map
  - Filter by zone (e.g. Jacksonville, Palm Coast, Daytona)

### 🧲 **Lead Finder Tool**
- Scrape Google Maps/Yelp for car/tire/tow businesses
- Highlight bad reviews = partnership opps
- Save to lead list (Replit DB)
- Export as CSV

### 💳 **Pricing & Payment Tracker**
- Manual pricing entries per job
- Auto-calculate quote suggestions using AI
- Display daily/weekly/monthly earnings
- Stripe support (future-ready)

### ⚙️ **Settings Page**
- Edit base pricing per zip/service
- Manage auto-reply toggle
- Set Fritzner’s phone number for forwarding
- Store Twilio credentials securely

---

## 🤖 AI INTEGRATIONS
### Together AI API (Free tier)
- Use `Mixtral` or `Mistral 7B` for smart replies, pricing logic, and lead prioritization
- Implement `POST /smart-reply` endpoint that takes lead info and returns:
  - Suggested reply message
  - Recommended service price
  - Urgency score (0–100)

### Twilio API
- Buy a new Twilio number ($1/month)
- Use it to:
  - Send SMS messages
  - Receive replies (optional webhook)
  - Forward calls to Fritzner’s main phone

### Email (optional backup)
- Use MailerSend or Email-to-SMS (e.g., 5551234567@vtext.com) as backup method
